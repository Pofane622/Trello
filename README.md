Trello Clone

A full-featured Trello clone built with React. Create boards, lists, and cards to manage your projects just like Trello.

Live Demo: https://unrivaled-strudel-39a0b2.netlify.app/

Features

- Create and manage multiple boards
- Add, edit, delete, and reorder lists
- Add, edit, delete, and reorder cards
- Mark cards as complete
- Filter cards by completion status
- Collapse and expand lists
- Copy and move lists
- Change list colors
- Table view for cards
- Dark and light mode toggle
- LocalStorage persistence
- Fully responsive

React, React Router, Context API, CSS, LocalStorage

Installation

git clone https://github.com/Pofane622/Trello.git
cd trello-clone
npm install
npm start
